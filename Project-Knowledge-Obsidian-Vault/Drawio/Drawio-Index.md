## How to use them for learning

1. Open one `.drawio` file in draw.io
2. Add code refs on edges (example: `gateway.py:929`)
3. Run a live command (`read` or `threshold 5000`)
4. Mark each hop green when confirmed in logs
5. Repeat until you can explain the whole path from memory
